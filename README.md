# 🧠 Memory Card Game

A simple and interactive Memory Card Game built using **HTML**, **CSS**, and **JavaScript**.

## Features

* 16 cards (8 matching pairs)
* Random card shuffle
* Move counter
* Timer
* Pairs found counter
* Restart game option
* Responsive design
* Smooth animations

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts

## Project Structure

```text
memory-card-game/
│
├── assets/
├── css/
│   └── style.css
├── js/
│   └── script.js
├── index.html
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Run `index.html` using **Live Server** or open it directly in a web browser.
